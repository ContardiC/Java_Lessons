# Java Lessons

Questo repository contiene lezioni e esempi di codice Java organizzati in vari pacchetti. Ogni pacchetto copre un argomento specifico e include codice di esempio per illustrare i concetti.

## Pacchetti

- **basicop**: Operazioni di base in Java.
- **collections**: Utilizzo delle collezioni in Java.
- **database**: Connessione e operazioni con database.
- **datastructures**: Strutture dati comuni in Java.
- **dates**: Gestione delle date e orari.
- **enumtype**: Utilizzo degli enum in Java.
- **exceptionhandling**: Gestione delle eccezioni.
- **inheritance**: Concetti di ereditarietà.
- **files**: Operazioni sui file.
- **geometry**: Esempi di geometria.
- **interface_examples**: Esempi di utilizzo delle interfacce.
- **lambdaexpression**: Utilizzo delle espressioni lambda.
- **patterns/mvc**: Implementazioni del pattern MVC.
- **socket**: Programmazione di socket.
- **strings**: Operazioni sulle stringhe.
- **swing**: Esempi di interfacce grafiche con Swing.
- **swing_new**: Nuovi esempi di interfacce grafiche con Swing.


## Struttura del Progetto

Ogni pacchetto contiene uno o più file Java con esempi di codice. È consigliato esaminare ogni pacchetto singolarmente per comprendere i concetti trattati.

## Come Usare Questo Repository

1. **Clona il repository**:
   ```sh
   git clone https://github.com/ContardiC/Java_Lessons.git
   ```

2. **Importa il progetto in un IDE** (come IntelliJ IDEA o Eclipse).

3. **Esplora i pacchetti** e esegui i file Java per vedere gli esempi in azione.
